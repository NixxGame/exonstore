Capabilities

Feature List

Survivor ESP

See all survivors through walls with customizable bounding boxes and skeleton overlays.

Killer ESP

Track the killer's position at all times with distance readout and direction indicator.

Bone / Skeleton Rendering

Full skeleton overlay on survivors and killers for precise positional awareness.

Distance Display

Real-time distance in meters displayed beneath every entity's name tag.

Snap Lines

Optional lines drawn from the center of the screen to each entity for quick scanning.

2D Minimap Radar

Compact overhead radar showing all entity positions relative to your own.

Customizable Colors

Per-entity color settings for boxes, skeletons, and snap lines via the in-overlay config.

Visibility Check

Color-coded indicator showing whether an entity is visible or obscured by geometry.

Generator Auras

All generators highlighted with real-time repair progress percentage displayed.

Hook Auras

All hooks shown on the overlay including state (empty, occupied, broken) and use count.

Chest Auras

Locate every chest on the map — opened and unopened — at a glance.

Totem Auras

All dull and hex totems revealed with a clear type label so you can prioritize cleansing.

Hatch Aura

Hatch location shown the moment it spawns so you always know the escape route.

Exit Gate Auras

Both exit gates highlighted as soon as generators are complete.

Pallet Auras

Standing and dropped pallets marked so you can plan looping routes in advance.

Item Auras

Ground items and dropped items visible through the map at all times.

Survivor Health States

Displays each survivor's current state: Healthy, Injured, Dying, or Hooked.

Hook Stage Tracker

See exactly how many times each survivor has been hooked including sacrifice progress.

Carried Survivor Indicator

Clear highlight when a survivor is being carried toward a hook.

Survivor Item Display

Shows the item each survivor is holding with an icon label next to their name.

Obsession Highlight

Obsession target always clearly distinguished from other survivors.

Killer Identity

Killer character name and power type shown in the HUD as soon as the match starts.

Killer Terror Radius

Visual ring showing the killer's current terror radius so you can react before hearing it.

Name Tags

Steam display names shown above each survivor so you always know who is who.

External Overlay

Renders entirely outside the game process — no injection, no DLL, minimal detection surface.

Hotkey Toggle

Bind any key to show or hide the overlay and individual feature categories on the fly.

In-Overlay Config Menu

Full settings panel accessible in-game — adjust colors, toggles, and distances without Alt-Tab.

Auto-Updater

Automatically fetches the latest build on launch so you're always running the newest version.

HWID Lock

License is locked to your hardware ID to prevent sharing and protect your account.

Smooth Rendering

60 FPS overlay rendering with interpolation for a clean, stutter-free visual experience.
